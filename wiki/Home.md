## 🚀 What is AI-SAST?

AI-SAST is a powerful, AI-driven static application security testing tool that supports both cloud-based (Google Vertex AI) and local open-source (Ollama) LLM backends.

### Key Features

- 🤖 **Flexible LLM Backends**: Vertex AI (Google Cloud Gemini) or Ollama (local models)
- 💬 **Detailed Reports**: HTML and markdown reports with actionable findings
- 🔄 **GitHub Actions Integration**: Automated scanning in CI/CD workflows
- 🎯 **Smart Filtering**: Configurable exclusion patterns
- 💾 **Feedback Loop**: Learn from developer feedback to reduce false positives
- 🔒 **Privacy Options**: Keep code on-premise with Ollama
